# 메이딘 브랜드 사이트 디자인 가이드라인

## 디자인 접근 방식
**참고 스타일**: DRS Korea (디알에스코리아) - 전문적이고 신뢰감 있는 기업 웹사이트 디자인

## 컬러 시스템
- **Primary Brand**: `#1a365d` (네이비 - 전문성/신뢰 상징)
- **Primary Light**: `#2d4a7c` (밝은 네이비 - 호버 상태)
- **Accent Blue**: `#7dd3fc` (하늘색 - 강조 포인트)
- **Background**: `#ffffff` (흰색)
- **Surface**: `#f9fafb` / `#f3f4f6` (회색 배경)
- **Border**: `#e5e7eb` (연한 회색 테두리)
- **Text Primary**: `#111827` (거의 검정)
- **Text Secondary**: `#6b7280` (회색 텍스트)

## 타이포그래피
- **폰트**: Pretendard (CDN)
- **계층구조**:
  - H1 (히어로): 3xl~5xl (36-48px), font-bold
  - H2 (섹션 타이틀): 3xl~4xl (30-36px), font-bold
  - H3 (카드 제목): lg~xl (18-20px), font-semibold
  - Body: base (16px), leading-relaxed
  - Small: sm (14px), text-gray-600

## 레이아웃 시스템
- **Spacing**: 모바일 py-20 (80px), 데스크톱 py-28 (112px)
- **Container**: max-w-6xl mx-auto px-4
- **Grid**: 모바일 1열, 태블릿 2열, 데스크톱 3-4열
- **Card Gap**: gap-6 또는 gap-8

## 섹션별 디자인 사양

### 헤더 (Header)
- 고정 상단 (fixed top-0)
- 흰색 배경 + 하단 테두리
- 좌측: MEIDIN 로고 (네이비 #1a365d)
- 중앙: 주요 메뉴 (회사소개, 제품소개, 제휴약국 찾기, 상담문의)
- 우측: 로그인 | 회원가입, 약사전용몰 버튼 (네이비 배경)
- 메뉴 구분선: 세로 1px 회색 라인

### 히어로 (Hero)
- **배경**: 네이비 그라데이션 (from-[#1a365d] via-[#234876] to-[#2d5a87])
- **레이아웃**: 2열 (좌: 텍스트, 우: 제품 이미지)
- **좌측 콘텐츠**:
  - 영문 서브타이틀 (하늘색)
  - H1 (흰색)
  - 설명 텍스트 (회색-200)
  - CTA 버튼: "제품 보러가기" (흰색 배경), "회사소개" (outline)
- **우측**: 제품 이미지 + 글로우 효과
- **하단 통계 바**: 회색 배경, GMP/100%/HACCP/ISO 인증 표시

### 회사소개 (#about)
- **배경**: 흰색
- **섹션 라벨**: "ABOUT US" (네이비)
- **3열 핵심가치 카드**: 원형 아이콘 배경
- **브랜드 철학**: 회색 배경 카드
- **연혁**: 네이비 세로 라인 + 원형 마커
- **핵심 수치**: 네이비 배경 + 흰색 텍스트

### 제품소개 (#product)
- **배경**: 회색 (gray-50)
- **섹션 라벨**: "PRODUCTS" (네이비)
- **메인 제품 카드**: 2열 레이아웃
  - 좌측: 제품 이미지 + 연한 네이비 배경
  - 우측: 제품 정보, 배지, 성분, 법정 문구

### 제휴약국 찾기 (#partners)
- **배경**: 회색 (gray-50)
- **검색바**: 흰색 카드 + 아이콘 입력창 + 검색 버튼
- **약국 카드**: 위치/전화/운영시간 정보
- **입점 문의 CTA**: 네이비 버튼

### 연구·기술 (#rnd)
- **배경**: 회색 (gray-50)
- **4단계 프로세스**: 카드 + 화살표 연결
- **연구소 이미지**: 와이드 이미지 + 캡션
- **인증 카드**: 3열 그리드

### 인증 및 등록 (#certifications)
- **배경**: 흰색
- **인증서 카드**: 클릭 시 모달로 원본 보기
- **카드 스타일**: 회색 배경, 호버 시 흰색

### 소식 (#news)
- **배경**: 회색 (gray-50)
- **뉴스 카드**: 날짜, 카테고리 배지, 제목, 요약
- **카테고리 색상**: 보도자료(네이비), 공지사항(노랑), 이벤트(초록)

### 상담문의 (#contact)
- **배경**: 흰색
- **2열 레이아웃**: 연락처 정보 + 문의 폼
- **연락처 아이콘**: 네이비 원형 배경
- **운영시간 카드**: 회색 배경

### 푸터 (Footer)
- **배경**: 네이비 (#1a365d)
- **텍스트**: 흰색/투명도 조절
- **링크**: 이용약관, 개인정보처리방침

## 컴포넌트 스타일

- **Primary**: bg-[#1a365d], text-white, hover:bg-[#2d4a7c]
- **Outline**: border-[#1a365d], text-[#1a365d]
- **Ghost**: text-gray-600, hover:text-[#1a365d]

### 카드 (Card)
- border-gray-200
- rounded (기본 radius)
- hover:shadow-lg transition-shadow

### 배지 (Badge)
- 카테고리별 색상
- border-[#1a365d] text-[#1a365d] (outline)
- bg-[#1a365d] text-white (filled)
