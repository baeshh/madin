Goal
네이처스팜 느낌을 참고하되, 쇼핑몰 기능 없는 기업 브랜딩용 단일 페이지를 React + Vite + TypeScript + Tailwind + shadcn/ui로 제작한다.
브랜드명: 메이딘 / 대표 제품: 센시비(참당귀 추출물로 염증 개선 영양제)

Tech Stack

Vite + React + TypeScript

Tailwind CSS + shadcn/ui + lucide-react (아이콘)

react-helmet-async (SEO 메타)

clsx, tailwind-merge (유틸)

배포 친화적 정적 SPA (앵커 내비게이션)

Create & Install

npm create vite@latest meidin-site -- --template react-ts

cd meidin-site

npm i -D tailwindcss postcss autoprefixer → npx tailwindcss init -p

npm i lucide-react clsx tailwind-merge react-helmet-async

shadcn/ui 설정

npm i -D @types/node

npm i tailwindcss-animate class-variance-authority @radix-ui/react-slot

프로젝트에 shadcn/ui 구성 파일 추가 및 Button/Card/Input/Form 등 기본 컴포넌트 설치

Tailwind 설정

tailwind.config.js에 다음을 반영:

content: ["./index.html","./src/**/*.{ts,tsx}"]

plugin: require("tailwindcss-animate")

theme.extend 색상 토큰:

brand: #2F8F6B, brand-2: #DCF2EA, acc: #B88C4A, surface: #F7F9F9, line: #E6ECEA

font: Pretendard 또는 Noto Sans KR 사용 (index.html에 CDN 링크)

Hero.tsx

About.tsx (#about)

3열 핵심가치 카드: 사람 중심 / 자연 중심 / 품질 중심

짧은 대표 인사말(2~3 문단 더미), 연혁 타임라인(예: 2019~2025 더미), 핵심 수치(연구인력/원료국가/인증 수)

RnD.tsx (#rnd)

프로세스: 원료선정 → 연구·검증 → 제조 → 품질관리 (4단계 카드)

인증/특허 배지(이미지) + 텍스트 설명(GMP/HACCP/ISO 더미)

실험실 이미지+캡션

Product.tsx (#product)

대표 제품 센시비 와이드 카드

상단 배지(“참당귀 추출물”, “컨디션 밸런스”, “간결 포뮬러”)

기능 요약 문장: 효능 단정 금지 톤 (“도움을 줄 수 있도록 기획되었습니다”)

제품 정보 블록: 원료/함량(더미), 섭취 방법(더미), 주의사항(법정 문구 박스)

추가 제품 2~4개용 자리 카드(더미)

Partners.tsx (#partners)

News.tsx (#news)

공지/보도자료 카드 리스트(날짜/카테고리 배지/간단요약/자세히 보기 더미)

Contact.tsx (#contact)

문의유형(제휴·납품/제품/기타), 이름/이메일/연락처/내용, 개인정보 동의 체크

onSubmit에서 필수값/이메일 형식 검증 → 일단 성공 알림. 서버는 없으므로 안내 메시지와 mailto: 또는 추후 API 연결 TODO 표기
